Instalación

Viene en el mismo plugin que deflate — una instalación, ambas direcciones. La salida siempre está en el idioma de tu entrada.

Consigue Claude Code

Si aún no lo tienes: npm install -g @anthropic-ai/claude-code, luego ejecuta claude e inicia sesión.

Agrega el marketplace e instala

Dentro de cualquier sesión de Claude Code:

/plugin marketplace add idlechara/bullshit
/plugin install bullshit@bullshit

Infla

Dale un hecho aburrido — opcionalmente con un nivel de bloat («suave», «90%», «slop máximo»):

/bullshit:inflate <un hecho simple y aburrido>

El recetario

Las mismas nueve señales que deflate detecta — leídas como instrucciones en vez de diagnósticos. Conoce el truco, usa el truco, detecta el truco.

La única regla

Nunca inventa hechos. Nada de números, métricas, empleadores, credenciales ni eventos fabricados — cada afirmación factual de la salida viene de tu entrada. La inflación es 100% envoltorio: paradojas, epifanías, moralejas y preguntas que cargan cero información, que es exactamente el chiste.

Esa es también la prueba de diseño: pásale la salida a deflate y deberías recuperar tu hecho original — y nada más. El cringe es la meta; el fraude queda fuera del alcance.
